Good to Know

HTML vs HTML5 vs CSS vs JavaScript:

  • HTML: The code that structures a webpage.
  • HTML5: The latest version of HTML with new cool features.
  • CSS: How you make your page look nice and colorful.
  • JavaScript: Adds interactivity to your webpage.

Div vs Span:

  • Div: A block-level container, takes the full width.
  • Span: Inline container, only wraps around the content you want.
